### Oslo Bysykkel Demo
An app for checking available bikes and locks at a given location

## Getting started
python3 oslobysykkeldemo.py

### Prerequisities
The program requires:  
requests==2.18.4  
PyQt5==5.11.2  

pip3 install requests PyQt5 --user  
or  
pip3 instal requests  
sudo apt install python3-pyqt5
